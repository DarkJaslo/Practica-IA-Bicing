# IA-Practica-Bicing
Práctica de búsqueda local para IA. Leer EnunciadoBusquedaLocal.pdf para más información.

Instrucciones de compilación:

### Linux:

Dar permisos de ejecución
<tt>chmod +x compila.sh</tt>
<tt>chmod +x ejecuta.sh</tt>

Compilar:
<tt>./compila.sh</tt>

Ejecutar:
<tt>./ejecuta.sh</tt>


### Windows:

Compilar:
<tt>compila.bat</tt>

Ejecutar:
<tt>ejecuta.bat</tt>
